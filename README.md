# Crime Analysis Using Chi-Square Test (R)

## Overview
This project analyzes crime data across multiple cities using a chi-square test of independence.

## Dataset
The dataset contains counts of violent and property crimes for five cities in Michigan.

## Method
A chi-square test was used to determine whether crime type distribution differs by city.

## Tools Used
- R
- Chi-square test of independence
- Matrix data structures

## Key Code
The analysis uses `chisq.test()` to compare observed vs expected frequencies.

## Goal
To practice statistical testing and understand relationships between categorical variables.
